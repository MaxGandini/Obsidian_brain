The kernel is a computer program at the core of a computer's operating system. 
![[b080063dbe7f9cf817972c558ef96e78-168305858.png]]
It is responsible of solving conflicts between different processes and is a portion of the operating system's code that is always resident in memory.
It is also responsible of facilitating interactions between hardware and software components, for example, via device drivers. 
It is the first program loaded on startup.
